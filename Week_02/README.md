二叉树的遍历：使用递归，或栈。
N叉树：前序和后序，注意入栈的顺序。
      前序以孩子节点的‘倒序’顺序入栈，（循环开始处，pop完，放入返回数组，在到孩子节点（逆序））
      后序入栈的顺序可以以 后序顺序的逆序入栈。（循环开始处，pop完，放入返回数组，再到孩子节点（正常list顺序））
    
  堆：数据流取（几个）最大最小值等问题

          2.process
